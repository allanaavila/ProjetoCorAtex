# ProjetoCorAtex

Projeto Cor
O Projeto Cor é uma aplicação desenvolvida em C# no ambiente Visual Studio, projetada para fornecer orientação aos pais no que diz respeito às configurações de dispositivos e jogos de seus filhos. Com a crescente presença de tecnologia na vida cotidiana, é essencial que os pais possam gerenciar e supervisionar as atividades digitais de seus filhos de maneira fácil e eficaz.

Principais características
Configurações de Dispositivos: Oferece uma interface intuitiva para personalizar as configurações de segurança em smartphones, tablets e computadores.

Controle de Jogos: Permite que os pais definam limites de tempo, restrições de conteúdo e outras configurações relacionadas aos jogos que as crianças podem acessar.

Relatórios Detalhados: fornece relatórios abrangentes sobre o uso de dispositivos e jogos, ajudando os pais a entenderem melhor o comportamento digital de seus filhos.

Alertas Personalizáveis: Notifica os pais sobre atividades específicas ou especificadas de configurações predefinidas, permitindo uma resposta rápida a situações particularmente preocupantes.
